Brief Introduction

The line fitting dynamic programming algorithm (segmented least squares algorithm) also known as piece-wise algorithm is implemented here. The above algorithm is a line fitting algorithm whose main focus lies in finding least or optimal error values after segmenting points and no of segmentations will be equal to number of lines formed. In this algorithm error is calculated for segments and the optimal value is found using recursion based on the error and then finding it's extreme value. In this algorithm we can have any number of segments which can give us the best OPT for all the points.
The algorithm is a dynamic programming algorithm which outputs no of segments and equations of the lines formed by these segments . We have set the slope to max possible integer value whenever we encounter a line with infinity slope. This is one of the main problems encountered while implementing the algorithm, the above solution will also help with visualization of algorithm.
